modular neural network:
    * model: computational module 
    * input and output: container



input : 
    * methods:
        ** init() # read data from datasets/databases
        ** norm() # normlize 
        ** process() # process the data 
        ** send() # to model
        ** visualize() # display a sample of the input
        ** raw() # get unprocessed input
        ** send() # send data as input to a model or another container
    * flow/pipeline: 
        ** init ()
        ** visualize ()
        ** preprocess()
        ** norm ()
        ** send()
output :
    * methods:
        ** init() # write data to this container        
        ** norm() # normlize 
        ** process()
        ** send() # to input container
        ** visualize() # display a sample of the input
        ** raw() # get unprocessed output
    * flow/pipeline: 
        ** init ()
        ** visualize ()
        ** preprocess()
        ** norm ()
        ** send()
    

container interactions:
    * container -> model:
        ** input -> model
        ** model -> output
    * container -> container:
        ** output -> input
